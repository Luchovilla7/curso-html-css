Módulo 2: Estilizando con CSS

Objetivo: Aplicar estilos CSS y mejorar el diseño de una página web.

  • ¿Qué es CSS y cómo aplicarlo?

    CSS (Cascading Style Sheets) es un lenguaje que describe la presentación de documentos HTML. Hay tres formas de aplicar CSS:

    1. CSS Inline

    <p style="color: blue; font-size: 16px;">Texto con estilo inline</p>

    2. CSS Internal (en el head)

    <head>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>

    3. CSS External (archivo separado)

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

/* styles.css */
p {
  color: blue;
  font-size: 16px;
}

  • Selectores básicos

    Los selectores CSS determinan qué elementos HTML serán estilizados:

    /* Selector de elemento */
p {
  color: blue;
}

/* Selector de clase */
.destacado {
  font-weight: bold;
  color: #ff6b6b;
}

/* Selector de ID */
#header {
  background-color: #333;
}

/* Selector descendente */
.container p {
  margin: 10px 0;
}

/* Selector de atributo */
input[type="text"] {
  border: 1px solid #ddd;
}

/* Pseudo-clases */
a:hover {
  color: red;
}

  • Propiedades de color, fuentes y fondo

    /* Colores */
.elemento {
  color: #ff6b6b;              /* Color de texto hexadecimal */
  color: rgb(255, 107, 107);   /* Color RGB */
  color: rgba(255, 107, 107, 0.5); /* Color RGB con transparencia */
}

/* Fuentes */
body {
  font-family: 'Arial', sans-serif;
  font-size: 16px;
  font-weight: bold;
  line-height: 1.5;
  letter-spacing: 1px;
}

/* Fondos */
.hero {
  background-color: #f8f9fa;
  background-image: url('imagen.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
}

  • Box Model

    El Box Model define cómo se estructuran dimensionalmente los elementos HTML:

    .elemento {
  /* Dimensiones del contenido */
  width: 300px;
  height: 200px;

  /* Padding (espacio interno) */
  padding: 20px;           /* Todos los lados */
  padding: 10px 20px;      /* vertical horizontal */
  padding: 10px 20px 15px 25px; /* arriba derecha abajo izquierda */

  /* Border */
  border: 2px solid #333;
  border-radius: 8px;
  
  /* Margin (espacio externo) */
  margin: 20px;
  margin: 10px auto;       /* centrado horizontal */
}

/* Box-sizing */
* {
  box-sizing: border-box; /* Incluye padding y border en width/height */
}

  • Posicionamiento

    CSS ofrece diferentes métodos de posicionamiento:

    /* Static (por defecto) */
.elemento-static {
  position: static;
}

/* Relative (relativo a su posición normal) */
.elemento-relative {
  position: relative;
  top: 20px;
  left: 20px;
}

/* Absolute (relativo al ancestro posicionado más cercano) */
.elemento-absolute {
  position: absolute;
  top: 0;
  right: 0;
}

/* Fixed (relativo al viewport) */
.elemento-fixed {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

/* Sticky (híbrido entre relative y fixed) */
.elemento-sticky {
  position: sticky;
  top: 0;
}

  • Conceptos Básicos de Flexbox

    Flexbox es un modelo de diseño unidimensional para distribuir y alinear elementos en filas o columnas.

    /* Contenedor Flex básico */
.container {
display: flex;
flex-wrap: wrap; /* Permite que los elementos se envuelvan */
}

/* Dirección del flujo */
.container {
flex-direction: row;         /* Por defecto: izquierda a derecha */
flex-direction: row-reverse; /* Derecha a izquierda */
flex-direction: column;      /* Arriba a abajo */
flex-direction: column-reverse; /* Abajo a arriba */
}

  • Alineación y Distribución

    /* Alineación horizontal (justify-content) */
.container {
justify-content: flex-start;    /* Inicio */
justify-content: flex-end;      /* Final */
justify-content: center;        /* Centro */
justify-content: space-between; /* Espacio entre elementos */
justify-content: space-around;  /* Espacio alrededor */
justify-content: space-evenly;  /* Espacio uniforme */
}

/* Alineación vertical (align-items) */
.container {
align-items: stretch;      /* Por defecto */
align-items: flex-start;   /* Inicio */
align-items: flex-end;     /* Final */
align-items: center;       /* Centro */
align-items: baseline;     /* Línea base del texto */
}

  • Ejemplo Práctico: Navbar con Flexbox

    /* HTML */
<nav class="navbar">
<div class="logo">Logo</div>
<ul class="nav-links">
  <li><a href="#">Inicio</a></li>
  <li><a href="#">Servicios</a></li>
  <li><a href="#">Contacto</a></li>
</ul>
</nav>

/* CSS */
.navbar {
display: flex;
justify-content: space-between;
align-items: center;
padding: 1rem 2rem;
background-color: #333;
}

.nav-links {
display: flex;
gap: 2rem;
list-style: none;
}

/* Responsive */
@media (max-width: 768px) {
.navbar {
  flex-direction: column;
  gap: 1rem;
}
}

  • Fundamentos de CSS Grid

    CSS Grid es un sistema de diseño bidimensional que permite crear layouts complejos de manera sencilla.

    /* Grid Container básico */
.grid {
display: grid;
grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
grid-template-rows: auto;
gap: 20px; /* Espacio entre elementos */
}

/* Áreas de Grid */
.grid {
grid-template-areas:
"header header header"
"sidebar main main"
"footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

  • Propiedades Avanzadas de Grid

    /* Patrones de Grid */
.grid {
/* Columnas automáticas con mínimo y máximo */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* Filas con altura mínima */
grid-auto-rows: minmax(100px, auto);

/* Alineación de elementos */
justify-items: center;
align-items: center;
}

/* Posicionamiento de elementos */
.item {
grid-column: span 2; /* Ocupa 2 columnas */
grid-row: 1 / 3;    /* De línea 1 a 3 */
}

  • Ejemplo Práctico: Layout Moderno

    /* HTML */
<div class="layout">
<header class="header">Header</header>
<nav class="nav">Navigation</nav>
<main class="main">Main Content</main>
<aside class="sidebar">Sidebar</aside>
<footer class="footer">Footer</footer>
</div>

/* CSS */
.layout {
display: grid;
grid-template-columns: repeat(4, 1fr);
grid-template-rows: auto;
gap: 20px;
grid-template-areas:
"header header header header"
"nav    main   main   sidebar"
"footer footer footer footer";
}

/* Responsive */
@media (max-width: 768px) {
.layout {
grid-template-columns: 1fr;
grid-template-areas:
  "header"
  "nav"
  "main"
  "sidebar"
  "footer";
}
}
