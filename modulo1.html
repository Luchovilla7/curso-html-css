Módulo 1: Fundamentos de HTML

Objetivo: Aprender la estructura básica de HTML y cómo funciona en la web.

  • ¿Qué es HTML y cómo funciona?

    HTML (HyperText Markup Language) es el lenguaje fundamental de la web que define la estructura y el contenido de las páginas web. Funciona mediante un sistema de etiquetas que el navegador interpreta para mostrar el contenido al usuario.

    <!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>

  • Estructura básica de una página web

    Toda página web tiene una estructura jerárquica básica:

    • DOCTYPE: Declara la versión de HTML (HTML5)
    • html: Elemento raíz que contiene toda la página
    • head: Contiene metadatos, títulos y enlaces a recursos
    • body: Contiene el contenido visible de la página

  • Etiquetas esenciales

    Las etiquetas básicas de HTML incluyen:

    <html>

    Define el documento HTML

    <head>

    Contiene metadatos y recursos

    <body>

    Contiene el contenido visible

    <meta>

    Define metadatos como charset y viewport

  • Etiquetas de texto

    HTML proporciona una variedad de etiquetas para formatear texto:

    <h1>Encabezado Principal</h1>
<h2>Subencabezado</h2>
<p>Párrafo de texto</p>
<strong>Texto importante</strong>
<em>Texto enfatizado</em>

    Los encabezados van desde h1 (más importante) hasta h6 (menos importante).

  • Listas

    HTML ofrece tres tipos principales de listas:

    
<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
</ol>


<ul>
    <li>Elemento</li>
    <li>Otro elemento</li>
</ul>


<dl>
    <dt>Término</dt>
    <dd>Definición</dd>
</dl>

  • Enlaces

    Los enlaces son fundamentales para la navegación web:

    <!-- Enlace básico -->
<a href="https://ejemplo.com">Visitar sitio</a>

<!-- Enlace que abre en nueva pestaña -->
<a href="https://ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

<!-- Enlace a correo electrónico -->
<a href="mailto:correo@ejemplo.com">Enviar email</a>

  • ¿Qué es la semántica en HTML?

    La semántica en HTML se refiere a utilizar etiquetas que describan el contenido de manera significativa para mejorar la accesibilidad y SEO.

  • Ejemplo de etiquetas semánticas

    Algunas etiquetas comunes incluyen:

    
<header>
    <nav>Menú de navegación</nav>
</header>
<main>
    <section>
        Sección de contenido
    </section>
    <section>
        Sección de contenido
    </section>
    <section>
        Sección de contenido
    </section>
</main>
 <footer>
    Pie de página
</footer>

  • Ventajas de usar etiquetas semánticas

    • Mejora la accesibilidad para lectores de pantalla.
    • Facilita el posicionamiento en buscadores (SEO).
    • Hace que el código sea más legible y estructurado.

  • Imágenes

    Las imágenes se insertan usando la etiqueta img:

    <!-- Imagen básica -->
<img src="imagen.jpg" alt="Descripción de la imagen">

<!-- Imagen con dimensiones -->
<img src="imagen.jpg" alt="Descripción" width="300" height="200">

<!-- Imagen responsive -->
<img src="imagen.jpg" alt="Descripción" style="max-width: 100%; height: auto">

  • Tablas

    Las tablas se utilizan para mostrar datos tabulares:

    <table>
    <thead>
        <tr>
            <th>Encabezado 1</th>
            <th>Encabezado 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Dato 1</td>
            <td>Dato 2</td>
        </tr>
    </tbody>
</table>

  • Formularios

    Los formularios permiten la interacción con el usuario:

    <form action="/enviar" method="POST">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="mensaje">Mensaje:</label>
    <textarea id="mensaje" name="mensaje"></textarea>

    <button type="submit">Enviar</button>
</form>
